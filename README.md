# poke-match
🃏 A Pokemon Themed Memory Match Project utilizing HTML5, CSS3, and JavaScript.

_Disclaimer: Please note that this project is for demonstration purposes only. All rights belong to their respective owners. I do not own any of this content._ 

## 👀Live Demo

Try the application live at https://memory-match.kevintolentino.com/

## 💻Technologies Used

- HTML5
- CSS3
- JavaScript

## 💡Features

- User can view game
- User can flip cards
- User can match two cards
- User can win game
- User can view game stats (accuracy, wins, attempts)
- User can reset game and play again
